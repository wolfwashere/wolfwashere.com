It's 3:43;

the sun is down,

there is no life

in this ghost town.

Empty apartments,

empty streets,

empty bedroom,

empty me.

The sun comes up

and day is new;

only me,

There is no sand,

no tumbleweed,

this is no film,

this is no dream.

There is no life

far as can see,

there is no you,

only me.
